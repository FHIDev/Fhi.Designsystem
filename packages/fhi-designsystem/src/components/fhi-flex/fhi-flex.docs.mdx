# Flex

Fhi-Flex er en layoutkomponent som gjør det enkelt å lage fleksible og responsive oppsett ved hjelp av CSS Flexbox. Den gir kontroll over retning, mellomrom, og innpakning av innholdselementer.

## Passer til

- Gruppering av elementer som passer sammen. 
- Lage fleksible og responsive oppsett som tilpasser seg ulike skjermstørrelser.
- Justere avstand mellom elementer for å skape visuell balanse.
- Organisere innhold i rader eller kolonner med enkel konfigurasjon.
