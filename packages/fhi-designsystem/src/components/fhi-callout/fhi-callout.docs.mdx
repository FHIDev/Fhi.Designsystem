# Callout
Callout brukes for å trekke brukerens oppmerksomhet til ekstra viktige beskjeder eller statusendringer i grensesnittet.
Den kan benyttes for å vise informasjon, suksess, advarsler eller feil.

## Passer til
- Å utheve viktig informasjon til brukeren
- Tilbakemeldinger om noe som er blitt gjort riktig eller galt

## Ikon

Ikon kan gjøre det enklere å skille mellom ulike typer beskjeder og raskt signalisere alvorlighetsgrad.
Det gir ekstra visuell støtte, særlig for brukere som ikke får med seg farger.

Ikon er ofte nyttig, men ikke alltid nødvendig. Ønsker du et roligere grensesnitt, kan det lønne seg å vise Callout uten.
Vurdér å utelate ikon dersom meldingen er enkel, nøytral og ikke krever ekstra oppmerksomhet, eller hvis mange Callouts vises samtidig.
Innholdet skal uansett være tydelig, uavhengig av ikon og farge.
