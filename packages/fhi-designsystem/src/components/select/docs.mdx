# Select
Select lar brukere velge fra en liste med alternativer.
Når man trykker dukker det opp en liste med de forshåndsdefinerte alternativene.

## Passer til

- Å velge fra en liste større enn det som passer å bruke Radio til
- Å velge fra en liste som ikke har plass i en gitt kontekst
