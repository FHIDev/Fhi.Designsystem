# Data Table

Data Table brukes for å presentere data og informasjon på en ryddig måte. Brukeren kan raskt få oversikt og skanne innholdet effektivt.

Tabellen (`<fhi-data-table>`) bygges opp av rader (`<fhi-data-table-row>`) med celler (`<fhi-data-table-cell>`).

## Passer til

- Å vise lister med større mengder strukturert data på en måte som er lett å lese og sammenligne.
