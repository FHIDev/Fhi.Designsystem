## Installasjon

### Komponentbibliotek
 
Kjør følgende kommando for å legge til komponentbiblioteket i prosjektet ditt:
```zsh
npm install @folkehelseinstituttet/designsystem
```
**eller**, om npm ikke er et alternativ, kan du bruke vår CDN. Legg til følgende script-tag i HTML-filen din:
```html
<script
  src="https://zealous-field-027caf703.5.azurestaticapps.net/fhi-designsystem.js"
  crossorigin="anonymous">
</script>
```

<div style={{padding: ".25rem 1rem", background: "#FEE9E6", borderRadius: "2px", borderBottom: "4px solid #EC7C73", marginBottom: "2rem"}} >
<p>
Vi anbefaler ikke å bruke CDN i produksjon.
Det krever at klienten laster ned hele biblioteket, uavhengig av hvilke komponenter som faktisk brukes. Dette påvirker ytelsen og båndbreddebruken negativt.
</p>
Bruk bare CDN hvis installasjon av pakken med npm ikke er et alternativ.
</div>

## Bruk

### Angular
Angular har innebygd støtte for webkomponenter. Du kan derfor bruke komponentene direkte i Angular-prosjektet ditt.
Dette er et eksempel på hvordan du bruker `fhi-button`-komponenten i et Angular-prosjekt:

Først må vi fortelle Angular at vi bruker webkomponenter.
Vi gjør dette ved å legge til <a href="https://angular.dev/api/core/CUSTOM_ELEMENTS_SCHEMA">CUSTOM_ELEMENTS_SCHEMA</a>
i angular komponent-filen sin moduldekoratør.
```ts
import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule],
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  bootstrap: [AppComponent],
})

```
Husk å importere komponenten i Angular-komponentens typescript-fil. I dette tilfellet importerer vi  `fhi-button`:
```ts
import '@folkehelseinstituttet/designsystem/fhi-button';
```

Nå kan du bruke komponenten som om det var et vanlig html-element i din Angular-komponents HTML-fil:

_obs: `handleClick` er brukt her som et eksempel. Du må selv lage en metode i Angular-komponenten som håndterer klikk-eventet._
```html
<fhi-button (click)="handleClick($event)">Click me!</fhi-button>
```
### React

<div style={{padding: ".25rem 1rem", background: "#FFE7BA", borderRadius: "2px", borderBottom: "4px solid #F1A541", marginBottom: "2rem"}} >
FHI Designsystem supports React version 19 and above. We do not offer support for older versions of React.
</div>

I React 19 og over kan du bruke webkomponenter direkte i JSX-koden din som om de var standard html-elementer.

Dette er et eksempel på hvordan du bruker `fhi-button`-komponenten i et React-prosjekt:

Først, i din React-komponentfil, importer komponenten du vil bruke. I dette tilfellet vil vi bruke `fhi-button`:
```jsx
import '@folkehelseinstituttet/designsystem/fhi-button';
```

Nå kan du bruke komponenten som om det var et vanlig html-element i din React-komponents JSX-fil:

_obs: `handleClick` er brukt her som et eksempel. Du må selv lage en metode i Angular-komponenten som håndterer klikk-eventet._
```jsx
<fhi-button onClick={handleClick(event)}>Click me!</fhi-button>
```
