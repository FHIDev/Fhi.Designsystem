## Kan jeg bruke komponentene i Angular Forms?

Ja det kan du!

Bruk angular sitt `ngDefaultControl` direktiv for Ã¥ koble komponentene til Angular Forms.
Dette er et standard Angular-direktiv som lar deg bruke web-komponenter som en del av Angular Forms.

e.g
```html
<fhi-text-input ngDefaultControl [(ngModel)]="modelValue" />
```
