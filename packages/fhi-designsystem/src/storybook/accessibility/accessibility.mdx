# Universell utforming

Nettsider i offentlig sektor i Norge skal følge WCAG-standeren for universell utforming på nett. Les mer på [uutilsynet.no](https://www.uutilsynet.no/wcag-standarden/wcag-standarden/86).

## Farger

Alle visuelle komponenter i designsystemet bruker fargetemaer som møter kontrastkravene. 

## Aria-label

Komponenter i designsystemet støtter bruk av aria-label for å navngi HTML-elementer når nødvendig, for eksempel på en ikon-knapp. 

```
<fhi-button type="button" aria-label="edit" icon-only>
    <fhi-icon-pencil></fhi-icon-pencil>
</fhi-button>
```

## Tastaturnavigering

Komponenter støtter tastaturinteraksjoner, slik som enter/space for å trykke på en knapp.
